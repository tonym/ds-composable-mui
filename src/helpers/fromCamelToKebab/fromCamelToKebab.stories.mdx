<h1>fromCamelToDash</h1>

Converts camelCase to kebab-case.

<h2>API</h2>

<h3>Import</h3>

Path import

```javascript
import fromCamelToKebab from '@uiproto/core/helpers/fromCamelToKebab';
```

Named import (use with Angular)

```javascript
import { fromCamelToKebab } from '@uiproto/core';
```

<h3>Signature</h3>

```javascript
fromCamelToKebab(value: string): string
```

<h3>Parameters</h3>

<ul>
  <li>
    <strong>value</strong> : The string to format.
  </li>
</ul>

<h3>Returns</h3>

<ul>
  <li>
    <strong>string</strong>
  </li>
</ul>

<h2>Examples</h2>

<h3>Format a string:</h3>

```javascript
const formattedString = fromCamelToKebab('callMeIshmael');

console.log(formattedString); // call-me-ishmael
```
